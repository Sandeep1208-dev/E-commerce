# Simple E-Commerce Website

A responsive e-commerce static website built with HTML, CSS, and Bootstrap 4.

## 🔗 Live Demo
[View Live](https://YOUR_USERNAME.github.io/ecommerce-website)

---

## 📸 Features

- **Responsive Navbar** with smooth scroll navigation
- **Image Carousel** for homepage banner
- **Services Section** — Fast Delivery, Money Back Guarantee, 24/7 Support
- **Products Section** — with interactive modals for Headphones & Laptops
- **Trending Blogs** section
- **Explore Deals & Offers** section with product categories
- **Special Offers Banner** with gradient design
- **Contact Us** section
- **Footer** with social media icons, links, and copyright

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 4.5.2
- Google Fonts (Roboto, Bree Serif)
- Font Awesome 5 (icons)

## 🚀 How to Run Locally

1. Clone the repository:
   git clone https://github.com/YOUR_USERNAME/ecommerce-website.git

2. Open the project folder:
   cd ecommerce-website

3. Open index.html in your browser — that's it!

---

## 📦 Deployment

This site is deployed using GitHub Pages.

To deploy your own copy:
1. Go to your repo → Settings → Pages
2. Set source to main branch
3. Your site will be live at https://YOUR_USERNAME.github.io/ecommerce-website
